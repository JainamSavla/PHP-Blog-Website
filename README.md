This project is a fully functional blog website built using PHP, MySQL, and Bootstrap. It includes an admin panel for managing blog posts, categories, users, and comments.

Features Included:
User Authentication (Login & Register)
Admin Panel (Manage Posts, Categories, Users & Comments)
Create, Edit, Delete, and View Blog Posts
Rich Text Editor for Post Creation
Image Upload & File Handling
User Roles & Permissions (Admin, Author, etc.)
Search Functionality (Find Posts by Title & Category)
Secure Database Connection (PDO)
Responsive & Mobile-Friendly Design (Bootstrap)
Like & Dislike System for Posts
